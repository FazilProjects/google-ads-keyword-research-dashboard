# AGENTS.md

## Project Context

This repository contains a static frontend portfolio project:

Google Ads Keyword Research & Campaign Structure Dashboard

The project is for Fazil Waseem, an aspiring Performance Marketer learning Google Ads, Meta Ads, lead generation, ecommerce marketing, and AI-assisted marketing systems.

## Boundaries

- Use HTML, CSS, and JavaScript only
- Do not add external frameworks or libraries
- Keep the project GitHub Pages compatible
- Do not connect to Google Ads, APIs, or external keyword tools
- Do not claim live search volume, CPC, competition, forecasts, or official Google data
- Keep claims honest and portfolio-focused

## Implementation Notes

- Main app shell: `index.html`
- Styling: `style.css`
- Local planning logic: `script.js`
- Planning docs: `docs/`

When adding features, keep the planning logic explainable and based on:

- Business category
- Campaign goal
- Target location
- Search intent
- Keyword groups
- Match type suggestions
- Negative keyword filtering
- Ad group structure

## Design Notes

- Clean premium dashboard UI
- Light background with dark text
- Careful use of blue, red, yellow, and green accents
- Cards, grids, badges, tables, keyword chips, and campaign blocks
- Responsive layout for desktop and mobile

## Review Checklist

- The page works by opening `index.html`
- Generate, export, and reset controls work locally
- No external assets or dependencies are required
- The copy does not imply affiliation with Google
- The dashboard reads as a keyword research and campaign planning portfolio project
