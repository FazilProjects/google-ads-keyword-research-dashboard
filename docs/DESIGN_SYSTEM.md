# Design System

## Direction

The interface should feel clean, premium, and professional, with a search marketing dashboard style. It should look credible for a performance marketing portfolio while staying lightweight and easy to scan.

## Visual Principles

- Light background
- Dark readable text
- Clean panels and cards
- Clear hierarchy
- Practical tables and planning blocks
- Keyword chips and badges for scanability
- Restrained use of brand accent colors

## Color Palette

Use Fazil's brand accent colors carefully:

- Blue: `#1a73e8`
- Red: `#d93025`
- Yellow: `#fbbc04`
- Green: `#188038`
- Dark text: `#17212f`
- Muted text: `#64748b`
- Border: `#dfe6ef`
- Light surface: `#ffffff`
- Soft surface: `#f6f8fb`

The accent colors should guide attention, not overpower the interface.

## Typography

- Use system fonts for speed and GitHub Pages simplicity
- Keep headings direct and professional
- Avoid oversized text inside dashboard cards
- Maintain strong contrast between labels, headings, and body copy

## Layout

- Use a constrained max-width content area
- Use grids for dashboard sections
- Use cards for input and output modules
- Use responsive single-column layouts on mobile
- Keep controls grouped by task

## Components

Recommended components:

- Header with project title and portfolio positioning
- Input panel
- Select fields
- Textarea for the business brief
- Budget and location fields
- Primary, secondary, and reset buttons
- Status badge for local planner disclaimer
- Keyword chips
- Intent cards
- Ad group table
- Campaign preview blocks
- Ad copy cards

## UX Notes

- The first screen should be the dashboard itself, not a marketing landing page
- Controls should be understandable without long instructions
- The dashboard should not imply that generated keywords are official Google data
- Export should produce a simple local text file
