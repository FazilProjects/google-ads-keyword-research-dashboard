# Project Brief

## Project Name

Google Ads Keyword Research & Campaign Structure Dashboard

## Positioning

This is a professional portfolio project for Fazil Waseem, an aspiring Performance Marketer learning Google Ads, Meta Ads, lead generation, ecommerce marketing, and AI-assisted marketing systems.

The project should show that Fazil understands how a short business brief can be translated into search marketing planning assets before a live campaign is launched.

## Core Goal

Build a static frontend dashboard that helps users create a Google Ads keyword research and campaign structure plan from simple inputs.

The dashboard should be useful for learning, presentation, and portfolio storytelling. It should not claim to be a real Google Ads tool or connected advertising platform.

## Primary User

A small business owner, marketing student, junior marketer, or portfolio reviewer who wants to see how raw business context can become a structured search campaign plan.

## Inputs

- Business brief
- Business type
- Campaign goal
- Target location
- Monthly budget

## Outputs

- Keyword ideas
- Search intent groups
- Ad group structure
- Match type suggestions
- Negative keyword ideas
- Campaign structure preview
- Starter ad copy ideas
- Exportable plan

## Planning Logic

The local JavaScript logic should be based on:

- Business category
- Target location
- Campaign goal
- Search intent
- Keyword groups
- Match type suggestions
- Negative keyword filtering
- Ad group structure

## Non-Goals

- No Google Ads API connection
- No Google Keyword Planner connection
- No live search volume, CPC, competition, or forecast claims
- No user accounts
- No paid media automation
- No external frameworks or libraries

## Success Criteria

- Looks like a polished, professional dashboard
- Clearly communicates portfolio and learning purpose
- Uses simple, explainable local planning logic
- Works on desktop and mobile
- Can be hosted on GitHub Pages
- Avoids exaggerated marketing claims
